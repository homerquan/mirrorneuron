# Mirrorneuron Engine

A real-time decision engine

Readme:

Run CEP:
sbt run

Test CEP:
sbt test

Package into a jar (include deps):
sbt assembly

Run an interactive console(REPL):
sbt "run -c"

Branches:

stage_release -> dev build
prod_release -> rc build
